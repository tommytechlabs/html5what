HTML5. The Next Chapter of the Web.

HTML is the behind the scenes code of the web. A huge percent of the population of websites on the web use HTML to do what they do. Since the beginning of the Interent, HTML has been around and has been updated ever since. HTML5 is the latest version of HTML. But, what makes it so special? Until HTML5, HTML has not been updated for the past decade and has led to companies creating plugins to fix problems HTML did not have. This was ok, except that you need to take risks of viruses infecting your computer from these plugins. HTML5 fixes this and many other features of the web. HTML5 makes the web modern, so that your website can have the best 3D graphics, to running it offline, while still keeping the amazing simplicity of HTML. With the newest version of HTML5, the web will be changed for good.

Here are some of the newest features to HTML5:

Semantics

In the past, it has been very hard to create a base structure for your website (for example: header, main, nav, footer, etc.). With HTML5 Semantics, you can create content that follows this structure with the new Semantic tags. This isn't all Semantics do. The new Semantics include new structural elements, media elements, internationalization, attributes, link relation types, form types, and microdata syntax for additional semantics. Semantics will make developers life easier.
